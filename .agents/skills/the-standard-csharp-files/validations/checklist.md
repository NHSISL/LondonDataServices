# C# Coding Standard — Files — Checklist

- [ ] tsc-csharp-files-001: File name is PascalCase (e.g., `Student.cs`, `StudentService.cs`).
- [ ] tsc-csharp-files-002: Partial class file uses dot-separated aspect notation (`StudentService.Validations.cs`).
- [ ] tsc-csharp-files-003: Sub-aspect partial class file uses two-dot notation (`StudentService.Validations.Add.cs`).
- [ ] tsc-csharp-files-004: No camelCase, all-lowercase, underscore, or hyphen word separators in the file name.
- [ ] tsc-csharp-files-005: No concatenated aspect names without dot separator (`StudentServiceValidations.cs` is forbidden).
