# ❌ Bad Commit Messages

```
fix                          # no category, no description
update stuff                 # vague, lowercase, no category
wip                          # work-in-progress must not be committed
added the student service    # past tense, no category, lowercase
FOUNDATIONS: added student   # past tense (must be imperative); no period
foundations: Add student.    # category must be ALL-CAPS
```
