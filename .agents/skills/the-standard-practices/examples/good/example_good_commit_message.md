# ✅ Good Commit Messages

```
FOUNDATIONS: Add AddStudentAsync implementation.
FOUNDATIONS: Add structural validation for AddStudentAsync.
FOUNDATIONS: Add exception handling for StudentDependencyException.
BROKERS: Add InsertStudentAsync to StorageBroker.
CONTROLLERS: Add PostStudentAsync to StudentsController.
TESTS: Add ShouldAddStudentAsync happy path test.
CONFIGURATIONS: Add GitHub Actions pipeline for build and test.
```

# Format

```
{ALL-CAPS CATEGORY}: {Imperative sentence starting with capital letter, ending with period.}
```
