# The Standard React + TypeScript + Vite — Files — Checklist

- [ ] Every file contains exactly one primary responsibility (tsr-files-001)
- [ ] Every broker file is named `{domain}{Kind}Broker.ts` (tsr-files-002)
- [ ] Every broker interface is named `i{domain}{Kind}Broker.ts` (tsr-files-002)
- [ ] Every foundation service is named `{domain}Service.ts` (tsr-files-002)
- [ ] Every view service is named `{domain}ViewService.ts` (tsr-files-002)
- [ ] Every page is named `{Domain}Page.tsx` (tsr-files-002)
- [ ] Every component file uses `.tsx` extension (tsr-files-003)
- [ ] Every non-rendering file uses `.ts` extension (tsr-files-004)
- [ ] No file is named `utils`, `helpers`, `common`, or `misc` (tsr-files-005)
- [ ] All files are placed in the correct layer folder under `src/`
