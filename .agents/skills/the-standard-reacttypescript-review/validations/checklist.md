# The Standard React + TypeScript + Vite — Review — Checklist

- [ ] Every review comment cites a rule ID (tsr-review-001)
- [ ] Architectural errors are flagged as ERROR and style issues as WARNING or INFO (tsr-review-002)
- [ ] No review comment expresses personal preference without a backing rule (tsr-review-003)
- [ ] Each comment identifies the Standard layer it affects (tsr-review-004)
- [ ] Suggested fixes are the smallest change that satisfies the violated rule (tsr-review-005)
- [ ] No PR that bypasses Standard layers is approved (tsr-review-006)
- [ ] React Doctor / React DevTools warnings are flagged with rule IDs (tsr-review-007)
- [ ] AI-generated or scaffolded code was evaluated against all Standard rules before approval (tsr-review-008)
