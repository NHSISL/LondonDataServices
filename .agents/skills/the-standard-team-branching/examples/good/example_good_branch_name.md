# ✅ Valid Branch Names

The following branch names comply with the `users/[username]/[category]-[entity]-[action]` pattern
and use approved categories from the category list.

| Branch Name | username | category | entity | action |
|---|---|---|---|---|
| `users/jsmith/FOUNDATIONS-Student-Add` | jsmith | FOUNDATIONS | Student | Add |
| `users/jsmith/MINOR-FOUNDATIONS-Student-Update` | jsmith | MINOR FOUNDATIONS | Student | Update |
| `users/jsmith/DATA-Student-AddMigration` | jsmith | DATA | Student | AddMigration |
| `users/jsmith/BROKERS-StudentStorage-Insert` | jsmith | BROKERS | StudentStorage | Insert |
| `users/jsmith/CONTROLLERS-Student-POST` | jsmith | CONTROLLERS | Student | POST |
| `users/jsmith/MINOR-FIX-StudentService-Validation` | jsmith | MINOR FIX | StudentService | Validation |
| `users/jsmith/INFRA-Build-Setup` | jsmith | INFRA | Build | Setup |
| `users/jsmith/CODE-RUB-StudentService-Cleanup` | jsmith | CODE RUB | StudentService | Cleanup |
| `users/jsmith/DOCUMENTATION-README-Update` | jsmith | DOCUMENTATION | README | Update |
