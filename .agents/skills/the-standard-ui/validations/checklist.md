# The Standard UI — Checklist

- [ ] ts-ui-001: All C# logic is in a ComponentBase class; .razor file contains markup only.
- [ ] ts-ui-002: Base class is named with the Base suffix (e.g., StudentComponentBase).
- [ ] ts-ui-003: Component has exactly one UI responsibility; no mixed concerns.
- [ ] ts-ui-004: No broker is injected or called from a component base.
- [ ] ts-ui-005: Pages are under Pages/; components are under Components/.
- [ ] ts-ui-006: No inline styles in any .razor file.
- [ ] ts-ui-007: No hard-coded string literals in component markup or code-behind.
